This is a simple music playback controller using ESP32 (seeedstudio ESP32C3 board) via BLE

Im using the seeedstudio XIAO_ESP32C3 board, connected with 5 push buttons for:
1. Volume up
2. Volume down
3. Play/Pause
4. Previous Track
5. Next Track

Tested on iOS
